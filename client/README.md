# Client Directory

This is a React application setup for the Room Design Assistant project.

## Steps to Setup
1. Clone the repository.
2. Navigate to the `client` directory.
3. Run `npm install` to install dependencies.
4. Use `npm start` to run the application.

## Technologies Used
- React
- React-Router
- Redux (optional)
- Axios
